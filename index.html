Stone Tools in Upper Norrland: A study of stone tools in Norrbottens county, Sweden

Description: This dataset was collected and constructed as part of the project Stone Tools in Upper Norrland (Sw: Bergartsredskap i Övre Norrland). Providing base documentation and descriptions of artefacts found in Norrbottens county, this project aimed to document and make information and data avaliable for stone tools found in this region. The database was produced by surveying the artefacts on site and digitalizing what documents were avaliable that provided information about where the artifacts were found. This database represents one of the first open-access data avaliable in northern Sweden focused on stone tools and can be reused by any researcher or archaeologists studying lithic artefacts in northern Fennoscandia. (2025-02-15)
